# Шпаргалка по записи Шрифта РЖЯ

Короткий справочник по символам, которые понимает анализатор. Документ
вспомогательный: основным источником значений остаются справочные таблицы
Шрифта РЖЯ и файл `src/data.js`.

## Как читается запись

- запись состоит из чередующихся **кадров** и **таймлайнов**;
- кадр описывает конфигурацию рук, ориентацию, локализацию и контакты;
- таймлайн описывает движение между кадрами;
- немануальные компоненты (лицо, плечи, голова) выделяются отдельно;
- пробелы и переносы при разборе игнорируются, но сохраняются для подсветки.

## Конфигурации рук

Конфигурация чаще всего задаётся одним символом. Несколько символов подряд
без скобок обычно означают разные руки или ориентации, а не одну конфигурацию.

| Символ | Значение |
| ------ | -------- |
| 1 | указательный палец |
| 5 | раскрытая ладонь |
| А | кулак |
| В | плоская ладонь |
| О | округлая кисть |
| С | полукруг |

## Ориентации

Ориентация пальцев и большого пальца записывается отдельными символами и
часто идёт сразу после конфигурации руки.

| Символ | Ориентация |
| ------ | ---------- |
| Э | кисть вертикально ребром |
| - | кисть параллельно полу вбок |
| " | кисть вертикально фасадом |
| = | кисть горизонтально фасадом |
| Ж | кисть параллельно полу, перпендикулярно корпусу |
| % | кисть горизонтально, ребром к полу |
| . | большой палец под кистью |
| Ё | большой палец над кистью |

## Направления движения

| Символ | Направление | Ось |
| ------ | ----------- | --- |
| Ь | вверх | вертикаль |
| _ | вниз | вертикаль |
| / | от себя | глубина |
| З | к себе | глубина |
| Б | влево от говорящего | горизонталь |
| Ю | вправо от говорящего | горизонталь |
| \\ | диагональ | диагональ |

## Контакты

| Символ | Тип контакта |
| ------ | ------------ |
| ! | ударить (резкое соприкосновение) |
| ; | тереть (трение) |
| Ц | смахнуть |
| * | коснуться |
| + | держать или щипать |

## Модификаторы движения

| Символ | Значение |
| ------ | -------- |
| ?? | шевелить или тереть пальцами |
| ? | согнуть пальцы |
| (!) | большая амплитуда |
| (;) | маленькая амплитуда |
| 9!0 | быстрое движение |
| 9;0 | медленное движение |

## Локализации

Локализации записываются в скобках или между маркерами `9` и `0`.
Примеры с головой: `(Ё)` лоб, `(Й)` нос, `(.)` подбородок. Примеры с телом:
`9Ё0` плечо ведущей руки, `9Й0` грудь по центру, `9..0` запястье.

## Немануальные компоненты

Немануальный блок обычно стоит перед основной записью жеста и описывает
выражение лица, движение бровей, глаз, головы и плеч. В анализаторе такие
компоненты выводятся отдельной строкой над кадрами.

## Неоднозначные символы

Символы `!` и `;` могут быть как контактом, так и модификатором движения.
Анализатор выбирает интерпретацию по контексту и показывает альтернативы в
блоке «Возможные последовательности».

## Примеры записей

- `(ЁЁ)*.Эо` — локализация «висок», контакт «коснуться», ориентация, конфигурация;
- `."Ч_Ё=7_Ё=Ч` — три кадра, разделённые таймлайнами;
- `1Э..Э1 / Д Я` — двуручный кадр и круговое движение.

Подробные таблицы значений см. в `src/data.js` и в справочнике Шрифта РЖЯ.
